URL Shortening API - Node.js Microservice

A Node.js server paired with a MongoDB database. It will accept a URL following the '/new' path and return an object with a randomly generated shortened number sequenced paired to that URL. Passing this number sequence onto the root path will redirect to the paired URL.

You can paste the returned "short" value into your browser URL bar.
    
Ex: https://afternoon-crag-81590.herokuapp.com/your-numbers-here
  


  
Sample test cases:

  
Passing a valid URL returns JSON with a paired short url - https://afternoon-crag-81590.herokuapp.com/new/http://stackoverflow.com

  
Accessing a short-url redirects to the paired site - https://afternoon-crag-81590.herokuapp.com/374212

  
Passing an invalid URL (here, no protocol), will return an error - https://afternoon-crag-81590.herokuapp.com/new/google.com
